# Project Brief

## Objetivo

Criar uma landing page pessoal para Rafa que comunique maturidade full-stack, uso estruturado de IA e capacidade de transformar problemas de produto em software confiavel.

## Escopo

- Definir posicionamento.
- Estruturar a Home como landing page unica.
- Definir direcao visual.
- Planejar specs granulares.
- Preparar base para futura implementacao.

## Fora de Escopo

- Implementacao do site.
- Escolha final de stack.
- Deploy.
- Criacao final dos cases completos.

## Conteudo

Frase central:

> Eu uso IA como parte de um processo de engenharia, nao como atalho: transformo contexto em especificacao, especificacao em testes, e testes em software que da para confiar.

## Criterios de Aceite

- O projeto tem uma tese clara.
- A direcao visual esta documentada.
- A Home/landing page tem uma estrutura inicial definida.
- As futuras specs serao granulares e rastreaveis.
- A implementacao usara Bun como runtime/package manager.
